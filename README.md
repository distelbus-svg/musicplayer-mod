# Melodix Music Player

An in-game music player for Minecraft (Fabric, client-side only). Play your own MP3 and WAV files or listen to the Minecraft soundtrack directly from a GUI without leaving the game.

**Why use Melodix?** Listen to custom music while building, exploring, or mining. The mod integrates playback into the HUD with a mini player, audio visualizer, and notification system. You can switch between your personal library and the vanilla soundtrack with one click.

## Features

- **Dual source playback** — play user MP3/WAV files or Minecraft soundtrack tracks
- **In-game GUI** — full-screen music player with playlists, search, and progress bar
- **Mini player** — movable/resizable overlay with controls, visible while playing
- **Album art** — displays embedded ID3v2 APIC tags from MP3 files
- **Audio visualizer** — live 32-bar frequency bars above the hotbar
- **Color themes** — Dark, Light, Ocean, Nether, End (configurable in settings)
- **Keybindings** — configurable hotkeys for play/pause, next/prev track
- **Default playlists** — "Music Disks" and "All In-Game Music" created on first launch

## Requirements

- **Loader:** Fabric Loader >=0.19.2
- **Minecraft:** >=26.1
- **Dependency:** Fabric API

## Installation

1. Install Fabric Loader for your Minecraft version
2. Download the Melodix JAR and place it in your `mods` folder
3. Launch the game and open the music player with the default keybind (M)

## Configuration

Config file: `.minecraft/config/musicplayer.json`

Options include default source, theme, notification toggles, and pause-in-singleplayer mode.
